# test_case_pt
Exercise C-2 solution.

## Depends
Required packeges:
```
g++
make
cmake
```

## Build programm
To build this program run commands:
```
mkdir build
cmake -B build/
make -C build/
```

## Run program
```
./build/test_case_pt
```

### Params to program
You can run this program with debug print
```
./build/test_sort -d
```

### Work with program
During the work you need to enter numbers greater than 0.
